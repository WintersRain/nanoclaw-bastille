# TDG Configuration

## Project Information
- Language: TypeScript
- Framework: Node.js (ESM, Discord.js, Docker containers)
- Test Framework: vitest

## Build Command
npm run build

## Test Command
npx vitest run

## Single Test Command
npx vitest run {file}

## Coverage Command
npx vitest run --coverage

## Test File Patterns
- Test files: *.test.ts
- Test directory: src/__tests__/
